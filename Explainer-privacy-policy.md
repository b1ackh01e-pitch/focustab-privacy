# Privacy Policy for Explainer

Last updated: April 30, 2026

Explainer does not collect, store, or share any personal or user data. The extension provides simple explanations for text you select on web pages using Chrome Built-in AI (Gemini Nano). All processing happens entirely on your device — no data is sent to external servers or shared with third parties.

The extension uses the following permissions:
- **activeTab** — to inject the tooltip UI on the current tab where you selected text
- **contextMenus** — to add the "✨ Explain" option to the right-click menu
- **storage** — reserved for future features like saving preferences locally on your device
- **languageDetection** — to detect the language of selected text using Chrome's built-in API

The only data processed is the text you select (up to 2000 characters), which is used solely to generate an explanation. No browsing history, personal information, or user actions are collected.
